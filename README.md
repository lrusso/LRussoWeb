# LRusso.com Website

Free and open source Website that you can clone and adapt, that includes:

* A portfolio for your projects.
* A lightweight AI that can assist you.
* A scraping bot that can check data for you.
* A Spider Solitaire designed in Phaser.js.
* A Mahjong game designed in Phaser.js.
* A Tetris game designed in Phaser.js.
